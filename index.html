Color Picker (Web Extension)

Easily select your desired color with this intuitive color picker tool, available as a Web Extension. Once you choose a color, its hex code is automatically displayed for quick reference. You can also copy the hex code to your clipboard with a single click, making it seamless to integrate the color into your projects. Perfect for designers, developers, and anyone working with color customization—this extension enhances your productivity by providing a simple and efficient way to select and utilize colors without leaving your browser. Enjoy the ease of color picking right from your toolbar!


Try out the Demo

# Features

  • Color Selection: Choose your desired color using an intuitive and easy-to-use color picker interface.
  • Instant Hex Code Display: The hex code of the selected color is automatically shown as you pick, eliminating the need for manual code search.
  • One-Click Hex Code Copy: Quickly copy the hex code with a simple click for seamless use in your design or development projects. Ideal for designers, developers, and anyone working with color customization, providing both accuracy and convenience.

# Installation Guide

Step 1 : Download the Extension

1. Download the color-picker tool

2. Extract the contents of the ZIP file, if applicable.


Step 2 : Open the Extensions Page in Your Browser

1. Open your preferred web browser.

2. Navigate to the Extensions page using the corresponding URL from the table below:

Browser Extension Settings Page
Google Chrome chrome://extensions/
Mozilla Firefox about:addons
Microsoft Edge edge://extensions/
Safari x-apple-system://com.apple.Safari.Extensions

3. Enable Developer mode if required (varies by browser).


Step 3 : Load the Extension

1. Click the Load unpacked button (or equivalent in your browser).

2. Select the folder containing the extension files (where `manifest.json` is located).


Step 4 : Verify the Installation

- The Color Picker icon should now appear in the browser toolbar.

- Click the icon to open the popup and Pick Colors. That's it :)
